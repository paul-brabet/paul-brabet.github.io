Grids and cats

2 March 2017

What is grid based design?

Designing a website based on a grid, the most popular format being 12 columns and however many rows deemed necessary. It simplifies working with elements.

Whats all the hype about responsive webpages?

So your website doesn't look like rubbish on a smartphone! Or any device really. You can't be proud of your pretty website if it turns out a hideous mess for 50% of viewers.

What is semantic structure?

Using tags logically in accordance to the way they were designed to be used. For example, use tag h1 as the biggest headline text on the page. Don't repurpose it via CSS to be used as a substitute for the p tag, and then redesign h4 as the biggest headline.
