HTML, CSS and DOM

2 March 2017

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?
HTML files are like the blueprints for the structure of a house. CSS files are the instructions for the decoration, like the colour of paint and where furniture might be placed. The DOM is like the brain of a robot builder interpreting these instructions - if you look into it you should see instructions identical to your instructions laid out in the HTML and CSS (except where the robot has found a small error like a typo and generously fixed it). Because this happens in the brain of the robot, changing the instructions in the DOM changes the output without changing the actual written instructions (i.e. the HTML and CSS files are unchanged).
