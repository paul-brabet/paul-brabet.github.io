Javascript language

31 March 2017

In the JavaScript language, describe all the use cases for each of:
Parentheses (): Parantheses can be used for related to grouping expressions e.g. var a = ( 5 + 2 ) * 7,
And containing expressions e.g. if (i < 10).
It can be used for containing parameters/arguments e.g. function (parameter1, parameter2),
Or none at all to maintain syntax e.g. function().
Brackets []: Brackets are used to contain an array.
They are used for retrieving properties from objects in bracket notation, e.g. object["property"] or object[0]
Similar to the above, brackets are used to retrieve elements from arrays, e.g. array[0] retrieves the first in the array
Braces {}: Used in objects to denote where the description of an object begin and end.
Used in conditionals, loops, and functions to contain statements. Statements are the bits where instructions are given, e.g. if (a) {then do what happens in the braces}.
Single vs double quotation marks '' "": Both are used for strings or wrapped around a property when using object notation. Single and double quotation marks can be used simultaneously, which is handy if you have quotes within a string. It may mean the difference from having to 'escape' quotaton marks (i.e. telling the computer to not treat the quotation marks as a string start/end but as plain text). For example 'Jeremy said "Javascript is great!"' will come across clearly whereas "Jeremy said "Javascript is great"" will be problematic.
